비포펫
- 반려견 산책 중개 및 실시간 라이브 트래킹 서비스
(사이드 프로젝트에서 정식 합류)

기간: 2024.09 ~ 2026.03

## Tech Stack
`Flutter` `Riverpod/GetX` `NaverMap API`

> **주요 업무:** 반려견 산책 중개 플랫폼의 핵심 모바일 애플리케이션 개발 총괄 및 서비스 아키텍처 설계
> 

> **프로젝트 개요: 견주와 도그워커를 매칭하고, 산책 진행 상황을 실시간으로 기록하는 O2O 서비스의 메인 프론트엔드 개발 및 안정화**
> 

> **실시간 O2O 예약 및 매칭 시스템 구축**
> 
> - 도그워커와 견주 간의 예약 프로세스를 최적화한 실시간 매칭 인터페이스 개발
> - 불필요한 네트워크 요청을 줄이고 데이터 동기화 지연을 최소화하기 위한 효율적인 상태 관리(State Management) 아키텍처 설계

> 
> **백그라운드 실시간 GPS 트래킹 시스템 개발**
> 
> - 산책 시 도그워커의 이동경로를 실시간으로 추적하는 GPS 라이브 트래킹 기능 구현
> - OS별(iOS/Android) 백그라운드 위치 권한 및 배터리 절전 모드 대응을 통해 백그라운드 상태에서도 끊김없는 위치 데이터 수집 환경 보장
> - GPS 오차 및 튀는 현상을 보정하기 위한 데이터 필터링 로직을 적용하여 이동 경로의 정확도 향상

> **산책 종료 후 데이터 시각화 자동 보고서 생성**
> 
> - 산책 중 수집된 GPS 좌표 데이터를 기반으로 이동경로(Polyline), 산책거리, 소요시간 등을 시각화한 지도 UI 구현
> - 산책 중 발생한 이벤트(배변, 급수 등) 기록을 타임라인 형태로 결합하여 견주에게 전달되는 최종 산책 보고서 생성 및 데이터 직렬화 시스템 개발

> **기술 리더십 및 개발 문화 개선**
> 
> - 크로스 플랫폼 기반의 확장성 높은 클린 아키텍처 도입으로 코드 재사용성 및 유지보수성 향상
> - 지속적인 코드 리뷰 프로세스 정립 및 CI/CD 파이프라인 자동화를 통한 배포 안정성 확보

> **핵심 기능 구현 및 API 연동**
> 
> - **지도(Map) API 연동:** 실시간 산책 경로 트래킹 및 시각화, 거리/시간 기반 데이터 처리
> - **결제 및 정산 자동화:** PG사 연동 및 도그워커 정산 프로세스 자동화 시스템 설계
> - **커뮤니티 기능:** 산책 사진 공유 및 피드 기능 구현으로 리텐션
